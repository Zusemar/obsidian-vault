#plan #review #middle-golang

6-недельный план повторения для уровня middle Go-разработчика. Каждая тема — одна сессия (~45–90 минут): прочитать заметку → ответить на вопросы самопроверки → написать код по памяти.

---

## Принцип повторений

```
Изучил тему → повтори через 1 день → через 3 дня → через 7 дней → через 14 дней
```

Используй колонки Kanban в Obsidian или просто отмечай даты рядом с чекбоксом.
